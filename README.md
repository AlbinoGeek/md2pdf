md2pdf
======

Convert a Markdown to PDF.

## Usage

    $ go get github.com/hayajo/md2pdf
    $ md2pdf --css=main.css target.md output.pdf

## Requires

[wkhtmltopdf/wkhtmltopdf](https://github.com/wkhtmltopdf/wkhtmltopdf)
